# SaaS Customer Support Ticket Dashboard

A simple SaaS-style customer support dashboard that allows support teams to view and filter support tickets.

## Features

- Ticket dashboard table
- Filter tickets by status
- Filter tickets by priority
- Search customers in real time
- Colored status and priority badges
- Dynamic ticket count

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

saas-support-dashboard
│
├── index.html
├── style.css
└── script.js

## How It Works

The dashboard displays support tickets in a table format. Users can filter tickets based on status and priority, search for customers, and see the number of tickets currently displayed.
